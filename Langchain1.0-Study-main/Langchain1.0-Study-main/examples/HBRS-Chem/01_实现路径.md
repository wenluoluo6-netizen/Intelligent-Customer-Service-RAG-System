# 化工厂危险行为预警系统 - 实现路径

## 📋 项目概述

**目标**: 基于LangChain + QwenVL的化工厂危险行为预警系统

**核心功能**:
- 实时监控摄像头画面
- 自动识别危险行为（未戴安全帽、未穿救生衣等）
- 结合安全规范知识库判断违规
- 实时预警通知

---

## 🎯 整体架构

```
┌─────────────────────────────────────────────────────────┐
│  摄像头群（海康威视）                                     │
│  ├─ 摄像头1: 临水作业区                                   │
│  ├─ 摄像头2: 高空作业区                                   │
│  └─ 摄像头3: 化学品操作区                                  │
└────────────┬────────────────────────────────────────────┘
             │ RTSP流
             ↓
┌─────────────────────────────────────────────────────────┐
│  动态抽帧服务（OpenCV）                                   │
│  - 运动检测：只在画面变化时抽帧                           │
│  - 节省70%存储和处理成本                                  │
└────────────┬────────────────────────────────────────────┘
             │ 图片
             ↓
┌─────────────────────────────────────────────────────────┐
│  AI分析引擎（QwenVL + LangChain）                        │
│  ├─ QwenVL: 视觉分析（识别人员、装备）                    │
│  ├─ RAG知识库: 检索安全规范                              │
│  └─ Agent: 决策判断是否违规                              │
└────────────┬────────────────────────────────────────────┘
             │ 预警
             ↓
┌─────────────────────────────────────────────────────────┐
│  预警通知                                               │
│  ├─ 钉钉/微信机器人                                      │
│  ├─ 短信通知                                            │
│  └─ 现场大屏显示                                         │
└─────────────────────────────────────────────────────────┘
```

## 🔧 技术选型

### 1. 摄像头
**选择**: 海康威视网络摄像头
- 型号: DS-2CD2T45D-I5（400万像素红外）
- 价格: ¥800-1200/个
- RTSP协议支持
- 内置移动侦测

### 2. 视觉分析
**选择**: QwenVL（通义千问视觉模型）
- API: 阿里云 DashScope
- 价格: ¥0.008/图（qwen-vl-plus）
- 支持中文
- 多模态理解

### 3. 框架
**选择**: LangChain 1.0
- Agent: 自动调用工具
- RAG: 向量知识库
- Checkpointing: 对话历史

### 4. 向量库
**选择**: Chroma
- 本地部署
- 轻量级
- 中文支持

### 5. 运动检测
**选择**: OpenCV + 帧差法
- 免费
- 实时性好
- 易于集成

## ⚠️ 风险和应对

### 技术风险
**风险1**: QwenVL识别准确率不足
- **应对**: 收集反馈，微调提示词，考虑多模型融合

**风险2**: 运动检测误报高
- **应对**: 调优阈值，结合海康内置移动侦测

**风险3**: 网络不稳定
- **应对**: 本地缓存，断线重连机制

### 业务风险
**风险1**: 用户接受度低
- **应对**: 充分培训，逐步推广

**风险2**: 成本超支
- **应对**: 先小规模试点（3-5个摄像头），验证效果后推广
